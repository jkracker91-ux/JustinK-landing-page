BONUS #1: Two Unfamiliar Linux Commands


Command 1: lsblk

Example usage:
$ lsblk

Description:
lsblk (list block devices) displays information about all available block
devices on the system, such as hard drives, SSDs, USB drives, and their
partitions, in a tree-like format. It shows device names, sizes, types, and
mount points. This would be useful when you plug in a new USB drive and need
to find its device name before mounting it, or when you need
to verify disk partition layouts before resizing or reformatting a drive, or really 
need to understand where exactly you are in the file tree.


Command 2: uptime

Example usage:
$ uptime -p

Description:
uptime shows how long the system has been running since its last boot, along
with the current time, the number of logged-in users, and the system load
averages for the past 1, 5, and 15 minutes. The "-p" flag displays the uptime
in a human-readable format (e.g. "up 2 hours, 15 minutes"). This is useful for
checking server availability and diagnosing whether a system has recently
rebooted unexpectedly, or for monitoring how long a machine has been running
without a restart during maintenance windows.
